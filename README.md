Chatbot Asistente en la Nube – Versión Lite
Descripción

Este proyecto corresponde al desarrollo de un chatbot asistente en la nube orientado a la atención automatizada de consultas ciudadanas frecuentes. El sistema funciona como primer nivel de atención, permitiendo responder preguntas comunes y registrar solicitudes básicas mediante una interfaz conversacional.

El chatbot se desarrolla como un proyecto académico–técnico, siguiendo buenas prácticas de ingeniería de software, automatización, QA y documentación.

Objetivos

Automatizar consultas frecuentes.

Reducir carga operativa humana.

Brindar atención 24/7.

Registrar solicitudes de manera estructurada.

Sentar bases para escalamiento futuro.

Alcance (Versión Lite)
Incluye:

Respuestas automáticas (horarios, requisitos, información general).

Registro conversacional de solicitudes.

Manejo de errores y fallback.

Integración con Telegram.

Persistencia en base de datos ligera.

No incluye:

Integraciones institucionales reales.

Validación de identidad.

Datos personales reales.

Atención multicanal productiva.

Tecnologías utilizadas

Lenguaje: Python

Base de datos: MySQL

Mensajería: Telegram Bot API

Infraestructura: Railway

Control de versiones: GitHub
